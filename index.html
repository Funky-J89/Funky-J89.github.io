Testautomation

In 2021 heb ik met Python en Selenium een soort spraakcomputer gebouwd, welke op basis van mijn stem opdrachten mijn werkagenda kon openen en voorlezen, een timer kon zetten en af laten gaan, moppen kon vertellen, basis zoekopdrachten op internet kon uitvoeren en de radio kon aanzetten. Binnen mijn werk als (Senior) tester bij VGZ heb ik ervaring opgedaan in het onderhouden van testautomatisering Visual Studio / Silk4Net op basis van C#. Ik heb voor de volledige set met financiele API's geautomatiseerde testen gebouwd op basis van randomised data, en ondersteuning geboden aan de andere backoffice teams om ditzelfde te doen. Daarnaast heb ik enkele testcases geautomatiseerd in Python Selenium voor het geautomatiseerd testen van onze webapplicatie. Ik heb de lead genomen in een project voor een overgang naar een nieuw testautomatisering framework, waarbij ik verantwoordelijk was voor het uitdenken en meebouwen aan het framework, het schrijven van de testscripts, en het begeleiden van mijn collega's bij het leren werken met het nieuwe framework.

SQL

In 2021 heb ik het certificaat 'Databases en SQL Foundation' gehaald bij Exin. Vervolgens heb ik ook nog meerdere leerpaden voor SQL doorlopen, waaronder via CodeAcademy en SoloLearn. Binnen mijn werkzaamheden als (Senior) tester heb ik ervaring opgedaan met het schrijven van PLSQL queries, en het werken met Oracle database structuren. Voor onze vier backoffice teams (ca 45 man) was ik aanspreekpunt op het gebied van complexe SQL-vraagstukken. Zodoende heb ik ervaring met gebruik van CTE's, subqueries, joins en het optimaliseren van queries.
